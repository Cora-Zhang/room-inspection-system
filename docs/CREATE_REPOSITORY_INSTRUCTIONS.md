### 步骤1：在GitHub上创建仓库

1. 访问：https://github.com/new
2. 填写仓库信息：
   - **Repository name**: `room-inspection-system`
   - **Description**: `机房巡检系统 - 企业级智能监控与运维管理平台`
   - **Public/Private**: 根据需要选择（推荐Public）
   - **⚠️ 重要**：不要勾选任何 "Initialize this repository with" 的选项
     - ❌ 不勾选 "Add a README file"
     - ❌ 不勾选 "Add .gitignore"
     - ❌ 不勾选 "Choose a license"
3. 点击 **Create repository** 按钮
